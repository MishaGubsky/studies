Тренинг от ISsoft для студентов

О тренинге

Тренинг разделён на три части. В первой рассматриваются базовая структура HTML, ее базовые концепции и принципы. Верстка сайта с использованием таблиц.

Далее рассматривается стилизация сайта с помощью CSS. Продвинутая верстка с использованием новейших стандартов.

В основном блоке будут изучаются базовые аспекты JavaScript, включая работу с объектами, функциями и классами языка. Так же рассматриваются различные JavaScript-фреймворки (jQuery, AngularJS, KnockoutJS) и UI-компоненты на основе jQuery UI и Bootstrap.

Целевая аудитория тренинга: 1. Программисты, изучающие JavaScript и JavaScript-фреймворки, с целью углубить или систематизировать имеющиеся знания.2. Специалисты, которые хотят переквалифицироваться на Front-end разработку.

После прохождения курсов Вы сможете не только понимать, что за магия твориться ниже, но сами тварить магию!

 
var i=10;
var f=function(n){
	if (n<=1){
		return 1;
	}
	else{
		return f(n-1)+f(n-2);
	}
}
while (i>0 || i===10) {
	console.log(f(i));
	i--;
}

Темы рассматриваемые на тренинге

HTML и верстка

Введение в HTML, HTML5. Синтаксис языка и разметка документа. Работа с текстом, изображениями, аудио и видео файлами. Таблицы и формы.

CSS. наводим красоту

Основы CSS, CSS3. Синтаксис и стилизация простых документов.

JS

Базовые аспекты JavaScript. Основы языка. ООП в JS. DOM и BOM. JQuery и Angular2.
